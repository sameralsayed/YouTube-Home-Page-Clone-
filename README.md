# 📺 YouTube Exact Replica (2026)

**Pixel-perfect static clone** of the official English YouTube homepage (https://www.youtube.com/).

## ✨ Features
🎯 **Exact match** – #0f0f0f dark theme, red #FF0000 accents, fonts, texts, layout  
📱 **Fully responsive** – desktop, mobile, tablet (Bootstrap 5 + sidebar toggle)  
🔥 **Smooth animations** – card hover lift, chip active state, infinite scroll  
🔍 **Real YouTube elements** – top nav with search + mic, left sidebar, chips row, video grid with duration  
♾️ **Interactive** – hover effects, fake infinite scroll, create button  
🖼️ **16:9 thumbnails** with exact duration overlay  
🔍 **SEO optimized** – meta tags, semantic HTML, fast loading  

## 🛠 Tech Stack
HTML5 • CSS3 • JavaScript • Bootstrap 5 • SVG
